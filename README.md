# Metronome

This project is a metronome application built using modern web technologies. It is designed to help musicians keep time while practicing.

## Technologies Used

- **Preact**: For building the user interface (lightweight alternative to React).
- **TypeScript**: For type-safe JavaScript development.
- **Vite**: For fast and efficient development and build tooling.
- **ESLint**: For maintaining code quality.
- **Prettier**: For consistent code formatting.
- **Tailwind CSS**: For utility-first styling.
- **Shadcn**: For pre-designed UI components.

Stay tuned for updates as the project evolves!
